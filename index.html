Real World Projects

Band Website

A website for a band which ended up being built with the bandzoogle web builder as it was the most efficient solution to the problem, offering a commission free store for merchandise, digital music distribution (through spotify, iTunes etc.) domain and hosting and technical support for a very reasonable price. Quick and easy to build but design is limited to templates. Quick turnaround was essential and the budget was limited so the benefits outweighed the limitations in this case.

Studio Website (under development)

 My first full frontend solution for a small business using HTML, CSS & Bootstrap. Solution is still under development but most of the key features have been implemented. This site uses a lot of bootstrap classes to help organise the content and make it more presentable. I have utilised some of the dropdown and expanding tools to tuck content away neatly so it can be called when required. This is particularly prevelant in the 'rehearsal' and 'contact' pages. To save building a complete full stack I am using typeform contact forms.
